# MERN Stack E-commerce Project

## Description

This is a full-stack eCommerce application built with the **MERN** stack (MongoDB, Express, React, Node.js). It allows users to browse products, add items to their cart, make purchases, and manage orders. The admin panel provides an interface for managing products, viewing orders, and handling user data.

### Key Features:
- **User Authentication**: Sign up, login, and account management.
- **Product Browsing**: View a list of products with filtering and searching options.
- **Shopping Cart**: Add, remove, and update items in the cart.
- **Order Management**: Place an order and view order details.
- **Admin Dashboard**: Manage products, users, and orders.
- **Payment Integration**: Integrated with a payment gateway (e.g., Stripe or PayPal).
- **Responsive Design**: Optimized for both desktop and mobile devices.

---

## Technologies Used

- **Frontend**: React, Redux, React Router, Axios
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens), bcrypt
- **Payment Integration**: Stripe or PayPal API
- **Deployment**: Heroku, Netlify, or Vercel for deployment (depending on frontend/backend setup)

## Installation

### Prerequisites

Before you begin, make sure you have the following installed on your system:

- [Node.js](https://nodejs.org/) - For the backend and frontend development.
- [MongoDB](https://www.mongodb.com/try/download/community) - Database to store products, user information, and orders.
- [Stripe API Key or PayPal API Key](https://stripe.com/docs/keys) - For payment processing.
